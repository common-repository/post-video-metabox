Add Video URL Metabox to the post format video and show the video player in the single post thumbnail which can click to play on the thumbnail.

== Description ==
Add the Video URL Metabox to the Wordpress Post Format Video.

Once added the Video URL along with post thumbnail. The Video Icon will appear in the post thumbnail of the main post.

User/Viewer can click the video icon to play the video.

Supported Youtube EMBED Video.

== Installation ==
You can install plugin by using wordpress plugins install page or directly download this plugin and upload to wordpress manually.

== Frequently Asked Questions ==
Will this plugin work on custom post type ?
- No, It work on the post only.
